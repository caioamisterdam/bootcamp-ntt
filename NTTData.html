Introdução ao HTML na Prática

[Voltar]

A estrutura de um HTML é composta de três partes principais: html, head e body.
Na Head, estarão elementos que serão carregados pelo navegador. No body, estará tudo aquilo que será mostrado ao usuário.

Algumas tags principais:

  • title: título da página, mostrado na aba do navegador
  • ol: lista ordenada, com números
  • ul: lista não ordenada
  • li: item de uma lista
  • input: cria uma campo para digitação
  • img: tag para colocar imagens
  • h: título
  • p: parágrafo
  • blockquote: bloco de citação
  • sup: texto superescrito
  • strong: texto em negrito
  • i: texto em itálico
  • u: sublinhado
  • mark: marcar texto
  • a: link
  • small: texto pequeno

Atributos de tags:

  • id: identifica a tag
  • type: define o tipo de valor que a tag imput receberá (text, number, color, etc)
  • style: estilo da tag, colocando elementos css como color, class, etc.
  • src: atributo para a tag img, onde colocamos o endereço url de uma imagem
  • width: define a largura
  • href: para colocar o endereço de um link
  • target: forma como você quer abrir um link (_blank: em uma nova aba, _self: na mesma página)
  • title: balãozinho com uma legenda do link
